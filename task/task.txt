Тестовое задание.
 
Команда должна выводить информацию в output в виде текстового сообщения, в котором должно содержаться: 
	getinfo:
		суммарный размер всех файлов до начала выполнение операции над файлами;
		суммарный размер всех файлов после выполнения операции над файлами;
	
	
	
Нужно реализовать команду, которая должна проходить по заданному каталогу, в котором находятся только файлы с изображением в формате JPG\PNG, и пересохранять их в формате JPG. Пересохраненные файлы должны быть в формате JPG но без указание расширения (как в примере каталога), с указанным качеством сжатия. 
	аналогичную информацию по отдельным сконвертированным файлам, 
	указание исходного формата;
	информацию о файлах, которые не попали под алгоритм сжатия;
	Некоторые файлы могут быть уже изначально в том качестве, к которому надо привести. Такие файлы без необходимости не трогать. Необходимость  определяться следующем правилом: если текущие качество файла выше, чем задано командой, то понижать качество, иначе ничего не делать с исходным файлом. 


получать на вход параметры: 
	путь;
	качество (десятичное значение от 10..100), по умолчанию 75;
  
проставить public static и тп


* - Выбор инструмента зависит от кандидата, если есть опыт работы с yii2 то желательно на нём, если нет, то на нативном PHP с использованием сторонних библиотек. В таком случае требуется, что бы приложение было кроссплатформенным и имело легкую инфраструктуру для его использование на продакшене.
